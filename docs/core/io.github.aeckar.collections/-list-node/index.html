ListNode

abstract class ListNode<Self : ListNode<Self>> : Iterable<Self> (source)

An element in a doubly-linked list.

Unlike in Java, this library does not provide a dedicated linked list class. Instead, list nodes are operated on directly.

A nullable property of this type whose value is null is considered an empty linked list. Iterates over the elements in the linked list, starting from and including this one.

    val nodes = link(values(), 1, 2, 3)
    println(nodes.toList())   // [1, 2, 3]
