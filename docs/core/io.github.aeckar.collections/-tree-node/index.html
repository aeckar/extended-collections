TreeNode

interface TreeNode<Self : TreeNode<Self>> : Iterable<Self> (source)

An element in a tree.

Unlike in Java, this library does not provide a dedicated tree class. Instead, tree nodes are operated on directly.

Instances may contain child nodes, but do not contain a reference to their parent.

    val tree = buildTree("5") { // Values indicate iteration order
        this += "0"
        this += "3".."2".."1"
        this += "4"
    }
    println(tree)               // 4 (root node)
    println(tree.treeString())  // 5
                                // ├── 0
                                // ├── 3
                                // │   └── 2
                                // │       └── 1
                                // └── 4
