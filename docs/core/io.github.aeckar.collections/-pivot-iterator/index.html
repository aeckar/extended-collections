PivotIterator

interface PivotIterator<out E, P : Comparable<P>, out V> : RevertibleIterator<E, P>

A revertible iterator over a sequence of elements, each of which is assigned some value.

Use when it is necessary to map both positional data and metadata to elements in a sequence by using revertible iteration.
